# Premium Catalog ✨ — Misi 11: Build Your Own API App

Aplikasi katalog produk e-commerce berbasis React Native (Expo) yang mengonsumsi REST API dengan arsitektur UI 3-state (Loading, Success, Error).

---

## 🚀 Fitur Aplikasi

- **🟢 Level 1 (Core):** Fetch data via `async/await`, error handling `try-catch-finally`, UI 3-state (Spinner, Pesan Error, Data Tampil), `FlatList` dengan optimasi kartu produk, dan tombol **Coba Lagi** (*retry*).
- **🟡 Level 2 (Pengembangan):** 
  - [x] 🔄 **Pull-to-Refresh:** Tarik layar ke bawah untuk memuat ulang data.
  - [x] 🔎 **Search Bar:** Input pencarian untuk filter produk secara lokal.
  - [x] 📄 **Layar Detail (Modal):** Tap produk untuk melihat deskripsi dan rating lengkap.
  - [x] 🗂️ **Filter Kategori:** Tombol *chips* horizontal dinamis dari JSON API.
  - [x] 🎨 **Empty State:** Tampilan ramah jika hasil pencarian tidak ditemukan.
- **🔴 Level 3 (Bonus):**
  - [x] ↕️ **Smart Sorting:** Urutkan produk berdasarkan Harga Termurah, Termahal, atau Abjad (A-Z).

---

## 🌐 API & Tech Stack

- **API:** FakeStore API (`https://fakestoreapi.com/products`) tanpa API key.
- **Tech Stack:** React Native (Expo SDK), JavaScript (ES6+), React Hooks (`useState`, `useEffect`), Native Fetch API.
